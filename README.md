# Awesome Claw [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated directory of OpenClaw ecosystem resources, skills, plugins, integrations, tools, and community projects.

OpenClaw is a personal AI assistant platform. This list helps people discover useful projects around the OpenClaw ecosystem.


Website: <https://awesome-claw.com>

X/Twitter: [@awesomeclawhub](https://x.com/awesomeclawhub)

Community-curated and independent. Not an official OpenClaw project unless explicitly stated by OpenClaw maintainers.
