# Security Policy

Awesome Claw is a curated directory. It does not run user-submitted code, but links may point to third-party projects.

## Reporting security concerns

Please report concerns by opening an issue if the concern is public and non-sensitive, such as:

- a link to a suspicious or compromised project
- malware-like behavior in a listed project
- misleading project claims
- phishing or impersonation
- broken links that now point somewhere unsafe

For sensitive reports, contact the maintainers privately before publishing details.

## Review posture

Entries are curated on a best-effort basis. Inclusion does not mean endorsement, security audit, or guarantee.

Maintainers may remove entries at any time if they appear unsafe, abandoned, misleading, spammy, or no longer relevant to the OpenClaw ecosystem.
